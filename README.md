# Personal site

Minimal single-page site built with React and Vite.

## Development

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (usually http://localhost:5173).

## Customize

Edit `src/site.ts` — name, bio, links, projects, and email live there.

## Build

```bash
npm run build
npm run preview
```

Static output is in `dist/` for deployment (Vercel, Netlify, GitHub Pages, etc.).
